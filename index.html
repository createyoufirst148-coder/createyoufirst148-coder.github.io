01. About

안녕하세요, 프론트엔드 개발자 김개발입니다. 저는 사용자 경험을 중심으로 웹 서비스를 설계하고 구현하는 일을 하고 있습니다. HTML, CSS, JavaScript를 기반으로 React를 주로 사용하며, 반응형 UI와 접근성을 고려한 화면 개발에 관심이 많습니다.

코드를 작성할 때는 재사용 가능한 컴포넌트 구조와 유지보수하기 쉬운 코드를 지향합니다. 최근에는 협업 도구 대시보드 프로젝트를 진행하며 TypeScript와 REST API 연동 경험을 쌓았고, 이전에는 스타트업에서 사내 관리 시스템과 랜딩 페이지를 담당했습니다.

새로운 기술을 배우는 것을 좋아하며, 팀과의 원활한 소통을 통해 문제를 함께 해결하는 개발 문화를 선호합니다. 앞으로도 더 나은 사용자 경험을 만드는 개발자가 되기 위해 꾸준히 성장하고 싶습니다.

  • JavaScript
  • TypeScript
  • React
  • HTML & CSS
  • REST API
